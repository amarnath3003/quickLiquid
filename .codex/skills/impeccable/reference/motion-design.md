# Motion Design

- Use motion sparingly and with purpose.
- Typical durations: 100-150ms for feedback, 200-300ms for state changes, 300-500ms for layout changes, 500-800ms for entrances.
- Prefer transform and opacity over layout properties.
- Use smooth easing curves instead of bounce or elastic motion.
- Respect reduced-motion preferences.
- Use staggered reveals carefully and keep them short.
