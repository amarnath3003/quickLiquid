# Color & Contrast

- Use OKLCH, not HSL.
- Build palettes from a brand hue, then tint neutrals toward it.
- Prefer a clear primary, a disciplined neutral scale, and semantic colors.
- Keep contrast high enough for body text, controls, and placeholder text.
- Avoid gray text on colored surfaces, pure black, and pure white.
- In dark mode, use lighter surfaces rather than shadows for depth.
- Use explicit semantic tokens instead of relying on transparency everywhere.
