# Typography

- Choose a distinctive font from the brief, not a default reflex.
- Use fewer sizes with stronger contrast.
- Keep line length around 65ch for body text when possible.
- Tune line-height to content width.
- Avoid the usual defaults like Inter, Roboto, Arial, and system stacks unless performance or context demands them.
- Use one font family if it can do the job; add a second only when contrast is truly needed.
